# Potluck - Nginx

An extension to the Potluck gem that provides control over the Nginx process and its configuration files
from Ruby.

## Installation

Add this line to your Gemfile:

```ruby
gem('potluck-nginx')
```

Or install manually on the command line:

```bash
gem install potluck-nginx
```
